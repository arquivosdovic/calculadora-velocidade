# Calculadora de Velocidade (Premiere)

Aplicação em React que calcula a velocidade necessária para ajustar a duração de um clipe no Premiere, com base no FPS.

## Instalação
```bash
npm install
npm run dev
```

## Build para produção
```bash
npm run build
```

## Deploy no Netlify
Build command: `npm run build`  
Publish directory: `dist`
